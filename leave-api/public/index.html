Authentication APIs

POST /api/v1/auth/login

User login with email and password. Returns JWT token.

GET /api/v1/auth/me

Get current user information (requires authentication).

POST /api/v1/auth/logout

Logout user (client-side token cleanup).

Leaves Management APIs

GET /api/v1/leaves

Get all leaves for current user.

POST /api/v1/leaves

Create a new leave request.

PUT /api/v1/leaves/:id

Update a leave request.

DELETE /api/v1/leaves/:id

Delete a leave request.

Manager APIs

GET /api/v1/manager/dashboard

Get manager dashboard statistics.

PATCH /api/v1/manager/leaves/:id/approve

Approve a leave request.

PATCH /api/v1/manager/leaves/:id/reject

Reject a leave request.
